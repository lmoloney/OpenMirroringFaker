## Format

Each ADR uses YAML frontmatter with the following fields:

```yaml
---
id: NNN
title: Short title
status: Proposed | Accepted | Deprecated | Superseded
date: YYYY-MM-DD
tags: [tag1, tag2]
---
```

Followed by sections: **Context**, **Decision**, **Consequences**.
